### Installation

1. *Clone the repository*

   git clone <https://github.com/Sampavi15/AIGuide.git>
   cd AIGuide
   

2. *Install dependencies*
   
   pip install -r requirements.txt
   

3. *Set your API key*
   
   export GEMINI_API_KEY=your_api_key_here

   set GEMINI_API_KEY=your_api_key_here
